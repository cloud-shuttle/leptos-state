Why Leptos State?

🎯

State Machines

XState-inspired finite state machines with hierarchical states, guards, and actions. Perfect for complex application logic.

⚡

Reactive Stores

Zustand-inspired reactive stores with middleware support. Simple, performant, and type-safe state management.

🔧

DevTools

Built-in visualization and time-travel debugging. Monitor state changes and debug with ease.

🚀

WASM Optimized

Performance optimized for WebAssembly. Fast, efficient, and perfect for modern web applications.

🛡️

Type Safe

Strong compile-time guarantees powered by Rust. Catch errors early and build with confidence.

📚

Well Documented

Comprehensive documentation, examples, and guides. Get up and running quickly.

Simple & Powerful

use leptos::*;
use leptos_state::*;

// Create a reactive store
create_store!(AppStore, AppState, AppState { 
    count: 0, 
    user: None 
});

// Use in components
#[component]
fn Counter() -> impl IntoView {
    let (state, set_state) = use_store::();
    
    let increment = move |_| {
        set_state.update(|s| s.count += 1);
    };
    
    view! {
        

            
"Count: " {move || state.get().count}

            
        

    }
}
